The site highlights my technical background, selected projects, engineering strengths, and contact information in a clean, responsive interface.

[ryankoch.me](https://www.ryankoch.me)

## Overview

This portfolio was designed to present my experience in a clear, polished, and recruiter-friendly format. It focuses on real-world engineering strengths, including cross-platform mobile development, frontend architecture, debugging, UI implementation, and building reliable applications.

The site includes:

* A concise professional introduction
* Skills grouped by engineering focus area
* Selected project cards with technologies and impact summaries
* Light and dark theme support
* Multiple color theme options
* Responsive layout for desktop and mobile
* Contact links for GitHub, LinkedIn, and email

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

Make sure you have Node.js and npm installed.

You can check your versions with:

```bash
node -v
npm -v
```

### Installation

Clone the repository:

```bash
git clone https://github.com/ryankoch13/reactjs-vite-portfolio.git
```

Navigate into the project directory:

```bash
cd reactjs-vite-portfolio/portfolio
```

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

The app should now be running locally at the URL shown in your terminal, usually:

```bash
http://localhost:5173
```

## Available Scripts

## Design Goals

The goal of this site is to feel professional, modern, and approachable while keeping the focus on the work itself. The design prioritizes readability, responsive spacing, strong project summaries, and a smooth user experience across screen sizes.

## Deployment

This project can be deployed with any static hosting provider that supports Vite builds, including:

* Vercel
* Netlify
* GitHub Pages
* Cloudflare Pages

To create a production build, run:

```bash
npm run build
```

The generated production files will be available in the `dist` directory.
